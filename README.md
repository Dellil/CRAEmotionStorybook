# CRA + Emotion + Storybook

1. clone this repo
2. yarn install
3. yarn start && yarn storybook

- @craco/craco ^6.1.0
- @emotion/react ^11.1.4, @emotion/styled ^11.0.0
- @emotion/babel-preset-css-prop ^11.0.0
- @storybook ^6.1.16
- storybook-preset-craco ^0.0.3
- webpack-merge ^5.7.3
